# JersApp

Welcome to the JersApp monorepo. This repository contains the mobile client, web application, and backend API server.

## Repository Structure

- [**mobile/**](file:///d:/Jerin/Projects/JersApp/mobile): The mobile application built using React Native.
- [**web/**](file:///d:/Jerin/Projects/JersApp/web): The web application frontend built with Next.js.
- [**server/**](file:///d:/Jerin/Projects/JersApp/server): The backend API server built with Node.js.

Refer to the README in each subdirectory for setup and run instructions.
